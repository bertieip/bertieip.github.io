Onboarding Guide: Human Resources

Getting Started

You have been selected as the Human Resources data holder. As such, we need you to complete the template we have provided.

This guide will help you to complete the data into the format needed to build this part of the company impact.

Finding and Requesting Data

As most information is internal, you should be able to complete the template from your own systems.

We will be asking for a list of employees for the period in question, and current employees for ongoing analysis.

For example, if the period of analysis is 01/01/2021 - 31/12/2021 then please add all employees who were at the company from 01/01/2021 to today.

We will not contact anyone who is no longer at the company.

Completing Data Template


Admin Data

The orange panel on the left hand side of the sheet gives you all your admin data.

Admin Data

  • Basic Data: This is the company name and the period of data analysis
  • Location Data: This is the location of offices to complete data input for

Progress Data

  • Progress: here you can track the progress of the data collection
  • Completed By: Once this data has been completed, add your email and completion data so that we can ask any questions upon verfication.


Employee Data

The locations and addresses will copy over automatically into the Primary Location column. All you need to do is fill in the usage data and the supplementary data associated with it.

Most data should be available within the HR system you're currently using

  • Email: This is the employee's company email address. We will use this to allow them to login to the platform, and enter their own data for onboarding.
  • First Name: The employee name is used to personalise the platform. They can change this later in the process.
  • Last Name: The employee name is used to personalise the platform. They can change this later in the process.
  • Start Date: Enter the date that the employee joined the company. If the employee joined the company before the period of analysis, and the start date is unknown, the use the start of the analysis period. For example, if the period of analysis is 01/01/2021 - 31/12/2021 then please use 01/01/2021 as the start date.
  • End Date: If the employee has since left the company, please enter the date that the employee left. Otherwise, please leave blank if the the employee is still at the company.
  • Primary Location: Although many employees have flexibility, please assign a primary location for each employee.

Frequently Asked Questions


What if I don't know the data?

Please use data as accurate as possible to complete this. Email is the most important component of the analysis, as we can then ask individuals to complete any information that is missing.

When adding dates, the month is more important than the day, so the 1st or last day of the month is sufficient in most cases.
