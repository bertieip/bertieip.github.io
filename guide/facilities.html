Onboarding Guide: Facilities

Getting Started

You have been elected as the data holder for facilities. As such, we need you to complete the template we have provided you. This guide will help you to complete the data into the format needed to build this part of the company impact.

Where do I find the data?

Electricity

Electricity is provided directly as a utility bill to the bill payer. This will come on a traditional utility bill (typically via a PDF). This bill will contain monthly usage data, the supplier, the tariff, and whether the tariff is low-carbon or not (typically referred to as renewable, green, or low-carbon).

If you do not get sent electricity bills, you can ask your landlord.

Larger offices will tend to break this down on a space basis (i.e. room or floor), and data will be available on request

Heating

Heating is provided on bills as kWh if delivered via a traditional combustion system, or by purchased heat. If the building uses an air management system, this is covered under electricity.

If you do not get sent heating bills, you can ask your landlord.

Larger offices will tend to break this down on a space basis (i.e. room or floor), and data will be available on request

Refrigerants

Refrigerants are a very important component of facilities data as a relatively small leak of the refrigerant can result in a large impact as the chemicals used in the systems have potent greenhouse warming potential.

Data is usually available from the landlord as it is a legal requirement to keep a record of any f-gas leaks or top-ups, and these are reported in F-gas ledgers.

Water

Water data is available from the landlord on a quartely basis. Often this is delivered via a whole building so will need to be apportioned on a space share basis.

File Template

The Facilities Template

Admin Data

The orange panel on the left hand side of the sheet gives you all your admin data.

Confirm the data in the Admin Data and Location Data sections is correct.

Progress: here you can track the progress of the data collection.

Completed By: Once this data has been completed, add your email and completion data so that we can ask any questions upon verfication.


Usage Data

The locations and addressed will copy over automatically to the location columns. All you need to do is fill in the usage data and the supplementaary data associated with it.

Electricity

  • Supplier: This is the supplier i.e. EDF
  • Tariff: This is speicifc tariff typeF
  • Low Carbon Tariff: Some electricity is s such you can select Yes/No/Unknown to reflect this.

Heating

  • Exists: As some buildings are heated by electricity of VRV systems, they won't be included in the heating section. Select No if this is the case
  • Type: If heating systems exist as separate entity then select Gas or District Heating here.
  • Low Carbon Gas: Some gas providers allow users to offset their gas at point of use. As such you can select Yes/No/Unknown to reflect this.

Water

  • Unknown: Water data can be hard to access. If this is the case, we can use benchmark data to predict water use over the year. If data is unavailable then select TRUE in the dropdown, otherwise select FALSE or leave blank

Refrigerants

  • Type: Select the type from the dropdown options given.

What if I don't know the data?

If the data isn't available we can use benchmark data to fill in the gaps. We would expect this data to be available for most organisations.

Frequently Asked Questions


What if we work in a co-working Space like a WeWork or similar?

In this instance, we can speak directly to the building or facilities management through the office manager. Please put us in touch with office management.

What if my data isn't from the 1st of the month to the end of the month?

Billing cycles vary, but we ask you to use the data that best corresponds to the month in question. This is typically based on which month the majority of the billing cycle falls into. If the billing cycle happens to be exactly half-way through the month, please use the lower half of the bill to decide the date.

What happens if my data is only available on a quarterly/bi-annual/annual basis?

In this instance, we would ask you to split it by the number of months the data is for:

  • Quarterly: split the data by 3 and add equally into each month
  • Bi-annual: split the data by 6 and add equally into each month
  • Annual: split the data by 12 and add equally into each month
