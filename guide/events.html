Onboarding Guide: Company Events

Getting Started

You have been selected as the data holder for Company Events in the office. As such, we need you to complete the events wizard to add details about events held in the Reporting Period.

You'll see the following task in your to-do list here: vso.alectro.io/todo-list

This guide will help you to enter the correct data to build this part of the company impact.

Finding and Requesting Data

First up, what do we mean by Events? By events, we are referring to major company events such as the Christmas party, Summer Party, or Company Conferences.

Individual social events such as weekly or monthly team socials will be assessed from expense data, and so you don't need to include these.

If a major event is also included in expenses, we will use the data submitted in this data input wizard, and disregard the expense information to avoid double counting.

As most of this data is internal, there is no need to request external data making this section quick and easy to complete.


Completing the Events Data

Completing the Data

Using the Template

Using the template Although we recommend using the Wizards directly, you may with to collect the data before uploading it (alternatively, you can upload the raw data directly). Click here for more information using the template →

You're given a range of options within the Onboarding Wizard. One of these is to "Download Template". Click this to download the template.

When you click to download the template, you will see a modal pop-up with Key Data and Location Data which you can then use to complete the Admin section of the template.

Add the relevant data into the Admin Section of the template (Step 1), and then complete the specific data (Step 2).

Once the data is completed, you can click "Upload Raw Data" to upload the data.

We'll do the rest from here, and we'll be in touch if we have any questions regarding the data.

Using the Onboarding Wizard

When you click to begin your task, you'll see the Events onboarding panel.

Step 1: Start Wizard

Click Start Wizard to enter the data

Step 2: Enter Data

For each Event, there are a range of data inputs required, and some are optional:

Associated Office
  • Office Location: Select the office location that "hosted" the event. If this is a company wide event, this will usually be the company HQ.
Basic Information
  • Event Name: this is simply an identifyer. For example, "Company Christmas party"
  • Event Date: this is the start date of the event.
  • Location: this is the location of the event. It can be a restaurant name, or a place name, for example, "Beckingham Palace, London".
  • Attendees: enter the number of attendees to the event. This can be a rough number rather than needing to be exact.
  • Event Description: Please give a brief description. For example: "Christmas Party held in December. Event included drinks and dinner in central London.""
Provisions
  • Meals provided per person: Meals = a course. For example, if this was a 3 course dinner, then the value would be 3. If over multiple days, please sum. i.e. Monday: lunch and then a 3 course dinner = 4, and Tuesday: Breakfast = 1, then the total is 4 + 1 = 5.
  • Drinks provided per person: Drinks = standard drinks (alcoholic and soft). Please approximate the number of drinks per person on average.
Supplementary Information
  • Event/Entertainment provided (optional): i.e. if there was a particular activity. White water rafting, or darts etc
  • Hotel room.nights associated (if applicable): This is to help us to calculate the hotel/stay impact. A room.night is measured on an individual basis. For example, 10 people staying in their own rooms for 2 nights each = 10 * 2. Likewise if 10 people are sharing 5 rooms for one night, then the room.nights = 5 * 1.
Adding Another Event

If you need to add another event you can click 'Add another event', or else click 'Finish' to return to the wizard homescreen

Step 3: Mark as Complete

You'll see a list of events that have been created in the wizard homepage. Once you're satisfied you've added all of your events, you can mark this data as complete.

The section will complete, and you can go back to your to-do list if you have other items to complete, else your work is done!

Your to-do list will now have a completed message in it.

Frequently Asked Questions


What if we had an event abroad?

Please add the event as normal. We will then be able to track flights through expenses and the information you have provided. As always, the more detail given in the description part of the template the better!
