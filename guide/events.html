Onboarding Guide: Company Events

Getting Started

You have been selected as the data holder for Company Events in the office. As such, we need you to complete the template we have provided.

This guide will help you to complete the data into the format needed to build this part of the company impact.

Finding and Requesting Data

By events, we are referring to major company events such as the Christmas party, Summer Party, or Company Conferences.

Individual social events such as weekly or monthly team socials will be assessed from expense data, and so you don't need to include these.

As most of this data is internal, there is no need to request external data making this template quick and easy to complete.


Completing Data Template


Admin Data

The orange panel on the left hand side of the sheet gives you all your admin data.

Admin Data

  • Basic Data: This is the company name and the period of data analysis

Progress Data

  • Progress: here you can track the progress of the data collection
  • Completed By: Once this data has been completed, add your email and completion data so that we can ask any questions upon verfication.


Events Data

All you need to do is fill in the yellow boxes, providing any supplementary data associated with it, and we'll do the rest.

For each Event, there are a range of data inputs required, and some are optional:

Basic Data
  • Event Name: this is simply an identifyer. For example, "Company Christmas party"
  • Event Start Date: this is the date of the event. For a one day event it is just the day of the event, but for multiple days it is the start date.
  • Location: this is the location of the event. It can be a restaurant name, or a place name, for example, "Beckingham Palace, London".
  • Event Description: Please give a brief description. For example: Christmas Party held in December. Event included drinks and dinner in central London.
Required Data
  • Attendees: i.e. how many employees attended the event. Approximate numbers are ok.
  • Meals provided per person: Meals = a course. For example, if this was a 3 course dinner, then the value would be 3. If over multiple days, please sum. i.e. Monday: lunch and then a 3 course dinner = 4, and Tuesday: Breakfast = 1, then the total is 4 + 1 = 5.
  • Drinks provided per person: Drinks = standard drinks (alcoholic and soft). Please approximate the number of drinks per person on average.
Optional/If Applicable Data
  • Event/Entertainment provided (optional): i.e. if there was a particular activity. White water rafting, or darts etc
  • Hotel room.nights associated (if applicable): This is to help us to calculate the hotel/stay impact. A room.night is measured on an individual basis. For example, 10 people staying in their own rooms for 2 nights each = 10 * 2. Likewise if 10 people are sharing 5 rooms for one night, then the room.nights = 5 * 1.
  • Main method of transport (if applicable): If transport was required and organised, please indicated this here. For example, was a coach hired for transport? Or was it all via train travel?

Because Events are usually organised internally, this template will hopefully be one of the simplest information inputs to complete.

Frequently Asked Questions


What if we had an event abroad?

Please add the event as normal. We will then be able to track flights through expenses and the information you have provided. As always, the more detail given in the description part of the template the better!
